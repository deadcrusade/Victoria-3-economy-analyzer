# Victoria 3 Economic Analyzer - Standalone Edition

## Quick Start (No Python Needed!)

This is a **standalone executable** - just double-click `Vic3_Analyzer.exe` to run!

### First Time Use:

1. **Double-click** `Vic3_Analyzer.exe`
2. **Browse** for your Victoria 3 save folder (usually: `C:\Users\YourName\Documents\Paradox Interactive\Victoria 3\save games`)
3. **Click buttons** to analyze your saves!

### Windows Security Warning

**If Windows Defender blocks the .exe:**

This is a **false positive** - the tool is safe!

**To allow it:**
1. Click "More info"
2. Click "Run anyway"

#### For Testing Mods:

1. Click **"Start Monitoring"** before playing
2. Play Victoria 3 (auto-saves are tracked)
3. Click **"Stop Monitoring"** when done
4. Click **"Open Charts Folder"** to see results!

#### Charts Generated:

- Price trends over time
- Price crash analysis
- Overproduction heatmap
- Building profitability
- Multi-playthrough comparison

### Troubleshooting

**"Directory not found"**
‚Üí Click Browse and select your save folder manually

**No visualizations**
‚Üí Need at least 2 save files to show trends

**Antivirus blocks it**
‚Üí It's safe! Add an exception or click "Run anyway"

**Nothing happens**
‚Üí Check the output window in the app for error messages

### File Locations

The tool creates these folders:
- `data/` - Tracked economic data (JSON)
- `visualizations/` - Generated charts (PNG)

You can delete these folders to start fresh!
